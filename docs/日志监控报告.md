## 新增组件

### 1. logger.py - 结构化日志系统

**功能**:
- ✅ 分级日志 (DEBUG/INFO/WARNING/ERROR/CRITICAL)
- ✅ JSON 结构化输出（便于分析）
- ✅ 文本格式输出（人类可读）
- ✅ 日志轮转（按天/按大小）
- ✅ 自动飞书告警

**日志文件**:
```
logs/
├── app.log        # 主日志（JSON，按天轮转）
├── error.log      # 错误日志（单独文件）
└── api.log        # API调用日志
```

**专用方法**:
```python
logger.api_call(endpoint, user_id, duration_ms, success)
logger.cache_access(key, hit, duration_ms)
logger.business_event(event, user_id, details)
```

---

### 2. monitor_api.py - 监控API

**接口**:
| API | 方法 | 功能 |
|-----|------|------|
| `/api/admin/logs` | GET | 查询日志（支持筛选） |
| `/api/admin/logs/stats` | GET | 日志统计 |
| `/api/admin/logs/errors` | GET | 错误日志 |
| `/api/admin/dashboard` | GET | 仪表盘数据 |
| `/api/admin/alerts/test` | POST | 测试告警 |

---

## 告警机制

**触发条件**:
- ERROR 级别日志自动触发
- 5分钟冷却期，避免重复告警

**告警内容**:
- 告警标题
- 错误详情
- 时间/模块/用户上下文

**配置**:
```bash
export FEISHU_ALERT_WEBHOOK="https://open.feishu.cn/..."
```

---

## 使用示例

```python
# 记录API调用
logger.api_call(
    endpoint="/api/leave/balance",
    user_id="user_001",
    duration_ms=50,
    success=True
)

# 记录业务事件
logger.business_event(
    event="create_adjustment",
    user_id="hr_001",
    details={'employee': '张三', 'amount': 1.5}
)

# 错误日志（自动触发告警）
logger.error("计算失败", exc_info=True, extra={'event': 'calc_error'})
```
