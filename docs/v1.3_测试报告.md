## 三、部署环境测试执行步骤

### 3.1 环境准备

```bash
# 1. 进入项目目录
cd /path/to/annual-leave-system

# 2. 确保依赖已安装
pip install -r backend/requirements.txt
pip install pytest pyjwt

# 3. 配置环境变量
cp backend/.env.example backend/.env
# 编辑 backend/.env 填入实际配置

# 4. 验证配置
python3 -c "from backend.config import FEISHU_CONFIG; print('Config OK')"
```

### 3.2 执行测试

```bash
# 方式1: 执行所有 v1.3 测试用例
python3 -m pytest tests/test_v1_3_features.py -v

# 方式2: 执行特定模块测试
python3 -m pytest tests/test_v1_3_features.py::TestFeishuAuth -v
python3 -m pytest tests/test_v1_3_features.py::TestServiceMonthsAutoCalculation -v

# 方式3: 生成测试报告
python3 -m pytest tests/test_v1_3_features.py -v --tb=short --html=report.html
```

### 3.3 回归测试

```bash
# 执行 v1.1 原有测试（确保新功能不影响旧功能）
python3 -m pytest tests/test_leave_calculator.py -v
python3 -m pytest tests/test_timezone.py -v
```
