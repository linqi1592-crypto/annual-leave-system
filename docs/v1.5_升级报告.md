## Redis 缓存支持

### 新增文件
- `backend/redis_cache.py` - Redis 缓存后端
- 修改 `cache.py` - 支持自动切换

### 使用方法
```bash
# 环境变量配置
export CACHE_TYPE=redis
export REDIS_URL=redis://localhost:6379/0

# 或使用内存缓存（默认）
export CACHE_TYPE=memory
```

### 特性
- ✅ 多实例共享缓存
- ✅ 自动序列化/反序列化
- ✅ 连接池管理
- ✅ 健康检查
