## 1. 项目概述

### 1.1 背景
公司员工通过飞书请假系统完成请假手续，需要一个独立的年假查询应用，让员工能实时查看自己的年假余额和请假明细。

### 1.2 目标
- 提供清晰的年假余额展示（剩余/已用/总额）
- 支持复杂的年假计算规则（法定+福利+结转）
- 展示请假明细时间轴
- 部署为飞书工作台应用

### 1.3 用户
全体员工（约X人，试点阶段3-5人）

## 3. 年假计算规则

### 3.1 法定年假（按社保工龄）

| 社保工龄 | 法定年假天数 |
|----------|-------------|
| < 12个月 | 0天 |
| 12 - 180个月 | 5天 |
| 180 - 240个月 | 10天 |
| > 240个月 | 15天 |

**计算公式**:
```python
if 工龄月 < 12:
    法定年假 = 0
elif 工龄月 <= 180:
    法定年假 = 5
elif 工龄月 <= 240:
    法定年假 = 10
else:
    法定年假 = 15
```

### 3.2 公司福利年假（按司龄）

| 司龄 | 福利年假天数 |
|------|-------------|
| < 12个月 | 6天（新人福利） |
| 12 - 180个月 | 1天 + 司龄年数 |
| >= 180个月 | 司龄年数 |

**计算公式**:
```python
if 司龄月 < 12:
    福利年假 = 6
else:
    司龄年 = 司龄月 // 12
    if 司龄月 < 180:
        福利年假 = 1 + 司龄年
    else:
        福利年假 = 司龄年
```

### 3.3 年假封顶

| 员工类型 | 封顶天数 |
|---------|---------|
| 一般员工（工龄<=240月） | 12天 |
| 老员工（工龄>240月） | 15天 |

**计算公式**:
```python
当年总额 = 法定年假 + 福利年假
if 工龄月 > 240:
    当年总额 = min(当年总额, 15)
else:
    当年总额 = min(当年总额, 12)
```

### 3.4 跨年结转

- **结转规则**: 上年剩余年假最多结转3天到次年
- **有效期**: 次年1月1日 - 3月31日
- **过期处理**: 3月31日后未使用的结转部分清零
- **手工调整**: 支持HR后台调整上年剩余（处理误操作）

**计算公式**:
```python
# 1. 计算系统上年剩余
系统上年剩余 = 上年总额 - 上年已用

# 2. 应用手工调整
调整值 = sum(adjust_amount for 记录 in 调整记录表 
            if 记录.year == 上年 and 记录.is_active == true)
调整后上年剩余 = 系统上年剩余 + 调整值

# 3. 计算可结转额度（最多3天）
结转年假 = 0
if 当前日期 <= 本年3月31日:
    结转年假 = min(调整后上年剩余, 3)
```

**场景示例**:
- 张三2025年系统计算剩余2天，HR调整+1天（漏计补录）
- 调整后剩余 = 2 + 1 = 3天
- 2026年可结转 = min(3, 3) = 3天（正好封顶）

- 李四2025年系统计算剩余4天，HR调整-1天（多计扣除）
- 调整后剩余 = 4 - 1 = 3天
- 2026年可结转 = min(3, 3) = 3天（正好封顶）

### 3.5 入职/离职折算

**规则**:
- 当年入职或离职，年假按自然月折算
- 最小单位: 0.5天
- 四舍五入

**计算公式**:
```python
if 当年有新变动(入职或离职):
    工作月数 = 计算当年实际工作月数
    当年总额 = round(当年总额 * 工作月数 / 12 * 2) / 2
```

### 3.6 已用年假计算

**有效请假** = 已通过的年假/事假 - 已撤回的年假/事假

**计算公式**:
```python
已通过 = sum(时长 for 记录 in 请假记录 
           if 记录.请假类型 in ['年假', '事假']
           and 记录.申请状态 == '已通过')

已撤回 = sum(时长 for 记录 in 请假记录
           if 记录.请假类型 in ['年假', '事假']
           and 记录.申请状态 == '已撤回')

已用年假 = 已通过 - 已撤回
```

### 3.7 最终余额计算

```python
总年假额度 = 当年年假总额 + 结转年假
剩余年假 = 总年假额度 - 已用年假
```
